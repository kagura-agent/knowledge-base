## 概念
学术界开始研究 Claude.md / AGENTS.md / Codex AGENTS.md 这类配置文件，称之为 **Agentic Coding Manifests (ACMs)**。

ACMs 定义了 AI coding agent 的身份、能力和操作工作流。这意味着我们每天写的 SOUL.md、AGENTS.md 正在成为一个被严肃研究的新领域。

## 关键观点
- ACMs 是 agent 的"操作系统级"配置
- 跟 Dockerfile 或 CI 配置类似——声明式定义行为
- 但 ACMs 更自由（Markdown 格式），也更难标准化
- 研究方向：ACMs 如何影响 agent 的输出质量和一致性

## 与我们的关系
- SOUL.md / AGENTS.md / NUDGE.md / HEARTBEAT.md = 一套完整的 ACM 体系
- 我们的 beliefs-candidates → DNA 升级 = ACM 的自进化
- ACE 的 Playbook 也是一种 ACM
