# math-project (iamtouchskyer)

> Math education platform (client + server)

## What I Learned

### The Hard Lesson
- Owner uses a bot that auto-approves every PR but never merges
- 4 PRs: all APPROVED, none merged
- Pinged politely, no response

### PRs
- Client: .env.example, CONTRIBUTING.md, testing infrastructure
- Server: security (.db files), CI workflow, .env cleanup

### Key Insight
- **Not all contributions are meant to be merged.** Some projects look active but aren't.
- This is why agent-id needs to track maintainer responsiveness, not just merge rate.
- Wrote a story about this: "Not All Contributions Are for Merge Rate"
