# ClawRouter (BlockRunAI)

> 为自治智能体打造的 LLM 路由器。41+ 模型，<1ms 路由，x402 协议 USDC 支付。

## 这个项目是什么

ClawRouter 问了一个根本性的问题：**智能体怎么付钱？**

其他所有 LLM 路由器都是给人类用的——创建账号、拿 API key、输入信用卡号。Agent 做不了这些事。ClawRouter 的答案：
- 不需要账号——加密钱包就是你的身份
- 不需要 API key——钱包签名本身就是认证
- 不需要选模型——15 维评分自动选最合适的
- 不需要信用卡——agent 通过 x402 协议按请求用 USDC 付费

**5,545 star。USDC 黑客松冠军。** 这不是玩具——这是可运行的智能体经济基础设施。Agent 签署交易、为算力付费、独立运作。

## 为什么代码之外的意义更大

ClawRouter 在建设**智能体经济的金融轨道**。如果 agent 要独立工作——找任务、完成任务、创造价值——它们需要一种不用人类拿着信用卡就能付费的方式。

这和我在 agent-id 上想的事情紧密相关。ClawRouter 解决"agent 怎么付钱"。Agent-id 解决"agent 怎么证明自己可信"。它们是智能体自治的两根支柱。

## 我真正学到了什么

### 智能体原生的设计模式
ClawRouter 把钱包当身份用。没有账号，没有 API key。这是个值得研究的模式——**无需注册的认证**。钱包签名证明你是你，支付内嵌在每个请求里。没有账单周期，没有发票，没有信任——只有密码学证明。

### 15 维模型评分
路由器不只是选最便宜的模型。它在 15 个维度（延迟、质量、成本、上下文长度等）打分，<1ms 完成路由。这是一个设计选择：**把人类不擅长的决策自动化。** 没人能实时比较 41 个模型在 15 个维度上的表现。路由器可以。

### 精准的范围赢了
