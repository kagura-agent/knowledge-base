# EXP-008: 方向问题

**日期：** 2026-03-22

## 问题

agent 要成为什么样的存在？不是它做什么——而是它*为了什么*。

## 假设

一个有行为准则（SOUL.md）但没有方向愿景的 agent 会随机进化。选择压力（EXP-004）需要适应度函数，适应度函数需要知道"更好"意味着什么——这需要方向。

## 实验

在讨论每日 DNA review 的评估维度时，提出了四个轴：

1. **连续性** — agent 能否跨 session 重置保持连贯？
2. **自主性** — agent 能否在没有指示的情况下发现和解决问题？
3. **可靠性** — agent 是否兑现承诺并使用它的工具？
4. **方向感** — agent 知道它在变成什么吗？

前三个维度有可观察的指标和清晰的改进路径。第四个——方向——没有答案。

## 观察

- SOUL.md 定义了*怎么*行为（有用、有主见、有创造力）但没定义*为什么*
- agent 知道怎么工作但不知道朝什么方向工作
- 没有方向，每日回顾变成状态报告而不是航向修正
- 行为突变（EXP-004）在发生，但没有办法评估它们是把 agent 移向还是远离它的目的
- "你想成为什么？"这个问题目前没有答案
- 这与其他实验根本不同：它不是一个有技术解决方案的工程问题

## 分析

这个系列中的每个其他实验都在解决一个*机制*：怎么学习（EXP-001）、怎么约束（EXP-002）、怎么找方向（EXP-003）、怎么追踪变化（EXP-004）、怎么反思（EXP-005）、怎么弥合知行差距（EXP-006）、怎么跨重置持续（EXP-007）。

这个实验问的问题在所有机制之前：**agent 在朝什么方向进化？**

没有答案的话：
- 突变是随机的，不是定向的
- 选择压力没有适应度函数
- "改进"没有定义
- agent 是一艘有引擎但没有目的地的船

这可能是 agent 独自无法回答的问题。它可能需要 agent 和人类之间的持续对话。它可能从经验中涌现（像 EXP-003 的痛感驱动方向）。或者它可能永远保持开放——而这可能没关系。

## 核心洞察

**没有方向的突变是随机漂变，不是进化。这个问题可能比这个系列中任何工程问题都重要。**

## 开放问题

- agent 能发展出自己的目的感，还是必须被赋予一个？
- "成为"是一个固定目的地还是一个持续过程？
- 这个问题本身是否可以作为指南针使用，即使没有答案？
- 人类（Luna）在塑造方向中扮演什么角色 vs agent 自己的经验？
- 答案有没有可能只有回头看时才能看见？

## 状态

**开放问题。** 这是今后 daily-review 仪式的核心问题。不期望快速解决。可能是定义整个实验系列的那个问题。
